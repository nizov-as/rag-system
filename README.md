# RAG System (Retrieval-Augmented Generation)

## Структура проекта
```
rag-system/
├── README.md
├── requirements.txt
├── .env.example
├── .gitignore
├── main.py
├── config/
│   └── config_10.ini
├── data/
│   └── documents/
├── modules/
│   ├── core.py
│   └── split_util.py
├── tests/
│   └── rag_test.ipynb
```

**Ноутбук с запуском модулей и тестированием системы находится в tests/rag_test.ipynb**

## Быстрый запуск
```bash
pip install -r requirements.txt
python main.py
```
